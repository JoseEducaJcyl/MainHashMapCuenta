# Contador de Palabras

## Descripción
Aplicación de consola en Java que cuenta la frecuencia de palabras en un texto separado por comas.

## Funcionalidades
- Leer texto introducido por el usuario
- Separar palabras por comas
- Contar cuántas veces aparece cada palabra
- Mostrar resultados ordenados

## Tecnologías
- Java
- HashMap
- Scanner
- Arrays
